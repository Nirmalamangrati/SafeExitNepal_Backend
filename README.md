# SafeExitNepal Backend

SafeExitNepal Backend is a MERN-based backend API for an emergency safety and disaster alert mobile application.

## Features

- User Authentication (JWT)
- SOS Emergency Alerts
- Incident Reporting
- Realtime Notifications
- Offline Sync Support
- Media Upload
- Location-Based Services

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer

## Run Project

```bash
npm install
npm run dev
```
